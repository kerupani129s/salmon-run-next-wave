# サーモンラン NEXT WAVE ブキ Changelog

注意:

- ブキ以外の変更は除く。
- メインウェポンの追加は除く。
	- スペシャルウェポンの追加は含む。
- 不具合の修正は除く。
- 射撃継続時間の値は諸説ありますが、ここでは 射撃継続時間 = (射撃継続回数 - 1) × 連射間隔 とします。
